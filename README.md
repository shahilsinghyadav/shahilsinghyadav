## How to Customize

1. **Update Content** - Edit component files in `src/components/`
2. **Colors** - Modify CSS variables in `src/index.css`
3. **Animations** - Adjust animation timings in `src/animations.js`
4. **Theme** - Toggle light/dark mode with the button in the navbar
